# Education Management System (UI Main)

ขั้นตอนการติดตั้ง (Setup Steps):
1. `git clone <repository_url>` (โคลนโปรเจกต์)
2. `cd ui-main` (เข้าไปที่โฟลเดอร์โปรเจกต์)
3. `npm i` (ติดตั้งแพ็กเกจทั้งหมด)

*หมายเหตุ: ไฟล์ `.env` ตั้งค่ามาให้เรียบร้อยแล้ว ไม่ต้องแก้ไขหรือเปลี่ยนอะไรเพิ่มเติม*

หลังจากติดตั้งเสร็จ สามารถรันโปรเจกต์ด้วยคำสั่ง:
`npm run dev`
